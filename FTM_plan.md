# 适配频率域二维复场的函数Tucker模型（FTM）完整训练流程
> **同一PDE方程、固定初边值条件、仅方程参数频率$\omega$变化**的二维复值物理场（实部+虚部双通道），输入为不同频率下的稀疏观测，输出为频率维度的核心张量序列与共享空间潜在函数

---

## 一、问题设定与核心定义
### 1.1 场景适配说明
本流程针对频率域扫频场景修改，核心替换规则：
1. 原论文的**时间维度$t$** 替换为**频率维度$\omega$**，序列为不同频率下的场数据
2. 原论文的三维空间数据替换为**二维平面空间$(x,y)$**，每个空间点包含实部$Re$、虚部$Im$两个通道的复值场
3. 保留原FTM的通用逼近能力与稀疏离网观测适配性，新增复场双通道与频率维度连续性约束

### 1.2 核心符号定义
| 符号 | 精准含义 |
|------|----------|
| $D = \{O^b\}_{b=1}^B$ | 总训练数据集，共$B$个独立样本；每个样本$b$对应一组固定初边值、不同频率下的二维复场观测 |
| $O^b = \{O_{\omega_1}^b, O_{\omega_2}^b, ..., O_{\omega_M}^b\}$ | 单个样本$b$的完整观测集合，$M$为该样本的观测频率点数，$\omega_m$为第$m$个采样频率 |
| $O_{\omega_m}^b = \{(x_n, y_n, \omega_m, Re_{n,m}^b, Im_{n,m}^b)\}_{n=1}^{N_m}$ | 频率$\omega_m$下的稀疏观测集合：$N_m$为该频率下的观测点数，$(x_n,y_n)$为二维连续空间坐标（支持离网/非规则采样），$Re_{n,m}^b/Im_{n,m}^b$为该坐标点的实部/虚部观测值 |
| $R_x, R_y$ | 二维空间$x$、$y$维度对应的Tucker秩（超参数，平衡表达能力与计算量） |
| $f_{\theta_x}(x)$ | $x$维度的潜在函数，由参数为$\theta_x$的神经网络（推荐浅层MLP）参数化；输入为连续坐标$x$，输出为$R_x$维向量 |
| $f_{\theta_y}(y)$ | $y$维度的潜在函数，由参数为$\theta_y$的神经网络参数化；输入为连续坐标$y$，输出为$R_y$维向量 |
| $\mathcal{W}_{\omega_m}^{Re}, \mathcal{W}_{\omega_m}^{Im}$ | 频率$\omega_m$下实部、虚部分别对应的Tucker核心张量，维度均为$R_x \times R_y$；二者共享空间潜在函数，仅核心张量独立 |
| $vec(\cdot)$ | 张量向量化操作，$vec(\mathcal{W}_{\omega_m}^{Re})$为长度$R_x \times R_y$的列向量 |
| $\otimes$ | 克罗内克积，$f_{\theta_x}(x) \otimes f_{\theta_y}(y)$输出长度为$R_x \times R_y$的列向量，与向量化核心张量维度严格匹配 |

### 1.3 FTM核心拟合公式（适配二维复场）
对任意观测点$(x_n, y_n, \omega_m, Re_{n,m}^b, Im_{n,m}^b)$，FTM的拟合目标为：
$$
Re_{n,m}^b \approx vec(\mathcal{W}_{\omega_m}^{Re})^{\top} \cdot \left( f_{\theta_x}(x_n) \otimes f_{\theta_y}(y_n) \right)
$$
$$
Im_{n,m}^b \approx vec(\mathcal{W}_{\omega_m}^{Im})^{\top} \cdot \left( f_{\theta_x}(x_n) \otimes f_{\theta_y}(y_n) \right)
$$

### 1.4 频率维度正则化项(可选)
针对频率维度的物理连续性（相邻频率的场解平滑变化），定义**按频率间隔加权**的频率维度正则项：
$$
TV_{\omega}(\mathcal{W}) = \sum_{m=2}^M \left( \left\| \frac{\mathcal{W}_{\omega_m}^{Re} - \mathcal{W}_{\omega_{m-1}}^{Re}}{\omega_m-\omega_{m-1}} \right\|_F^2 + \left\| \frac{\mathcal{W}_{\omega_m}^{Im} - \mathcal{W}_{\omega_{m-1}}^{Im}}{\omega_m-\omega_{m-1}} \right\|_F^2 \right)
$$
作用：约束核心张量在频率维度的平滑性，避免频率间解的跳变；当频率采样为 `log` 或不等间隔时，该形式比未加权差分更稳定。若频率响应存在尖峰（高Q共振），可将二次差分替换为Huber/Charbonnier惩罚以减少过平滑。

## 二、FTM完整训练全流程
### 步骤1：训练数据准备与预处理
#### 1.1 数据集格式标准化
1. 总数据集构建：收集$B$组独立的扫频样本（每组样本对应固定初边值、不同频率$\omega$的二维复场观测），形成总数据集$D = \{O^b\}_{b=1}^B$
2. 单样本格式整理：每个样本$b$整理为频率维度的观测集合$O^b = \{O_{\omega_1}^b, ..., O_{\omega_M}^b\}$，支持不同样本的频率点数$M$、频率采样间隔不同

当前设置：$B=200$, $M=50$, 频率范围[2,51] 

3. 单频率观测整理：输入形态
   <!-- - 形态A（离网）：结构化观测点列表，每行`[x坐标, y坐标, 频率值, 实部观测值, 虚部观测值]` -->
   - 形态（网格稀疏）：稠密张量 `data: (B,M,H,W,2)` + 掩膜 `mask: (M,H,W,2)` 或 `(B,M,H,W,2)`
4. 坐标与归一化：对实部、虚部分别做全局min-max或z-score归一化；频率坐标与空间坐标建议归一化至[0,1]。

#### 1.2 超参数预设
提前固定以下超参数，完成训练配置：
| 超参数 | 推荐设置与说明 |
|--------|-----------------|
| $R_x, R_y$ | 二维空间的Tucker秩，目前44 |
| $\beta$ | 频率维度正则化系数，推荐1e-4~1e-2，控制频率平滑强度 |
| $\lambda$ | 核心张量Ridge系数（可选），推荐1e-8~1e-4，用于病态观测下增强数值稳定性 |
| 潜在函数结构 | $f_{\theta_x}, f_{\theta_y}$推荐使用3~5层MLP，激活函数用ReLU/Siren，输入维度1，输出维度对应$R_x/R_y$ |
| 优化器配置 | 潜在函数优化推荐Adam优化器，学习率1e-6，batch_size按需设置 |
| 收敛阈值 | 损失下降阈值1e-6，最大迭代轮数20000轮 |

### 步骤2：构建适配场景的FTM训练目标函数
完整损失函数由**双通道重构损失**和**频率维度正则化损失**两部分构成：
$$
\mathcal{L}_{FTM} = \mathcal{L}_{recon} + \beta \cdot TV(\mathcal{W})
$$

#### 2.1 双通道重构损失
对全数据集所有观测点的实部、虚部分别计算MSE损失，仅在有真实观测的点上计算，完美适配稀疏采样：
$$
\mathcal{L}_{recon} = \mathbb{E}_{(x,y,\omega,Re,Im)\sim D} \left[ \left\| Re - \hat{Re} \right\|_2^2 + \left\| Im - \hat{Im} \right\|_2^2 \right]
$$
其中$\hat{Re}, \hat{Im}$为FTM的拟合值，对应步骤1.3的核心公式。

#### 2.2 频率维度正则化损失
即步骤1.4定义的$TV_{\omega}(\mathcal{W})$，约束核心张量在频率维度的连续性，$\beta$为正则化权重。

### 步骤3：交替方向迭代优化（核心训练步骤）
FTM的待优化变量分为两组：
- 组1：共享的空间潜在函数参数$\theta_x, \theta_y$
- 组2：所有样本的实部/虚部核心张量序列$\{\mathcal{W}^{Re,b}, \mathcal{W}^{Im,b}\}_{b=1}^B$，其中$\mathcal{W}^{Re,b} = \{\mathcal{W}_{\omega_1}^{Re,b}, ..., \mathcal{W}_{\omega_M}^{Re,b}\}$

两组变量相互耦合，采用**交替方向迭代策略**：固定一组变量，优化另一组，循环执行直到收敛，是张量分解领域的稳定高效优化方法。

#### 子步骤3.1 固定潜在函数，优化所有样本的核心张量序列
1. **固定前提**：当前轮次的潜在函数$f_{\theta_x}, f_{\theta_y}$完全固定，对任意空间坐标$(x,y)$，可直接计算得到固定向量：
   $$
   \Phi(x,y) = f_{\theta_x}(x) \otimes f_{\theta_y}(y)
   $$
2. **问题转化**：重构损失退化为关于$vec(\mathcal{W}_{\omega_m}^{Re})$和$vec(\mathcal{W}_{\omega_m}^{Im})$的**线性最小二乘问题**，实部、虚部独立求解，共享设计矩阵$\Phi(x,y)$
3. **线性系统求解**：加入频率维度正则后，该问题可写成块三对角稀疏线性系统
   $$
   H\,w = b
   $$
   其中$w=[vec(\mathcal{W}_{\omega_1});\dots;vec(\mathcal{W}_{\omega_M})]$。工程上可用稀疏直接法或共轭梯度稳定求解（等价于该二次目标的全局最优解），无需对核心张量做梯度下降迭代。
4. **输出**：所有样本的核心张量序列$\{\mathcal{W}^{Re,b}, \mathcal{W}^{Im,b}\}_{b=1}^B$完成更新

#### 子步骤3.2 固定核心张量序列，优化潜在函数参数
1. **固定前提**：当前轮次的所有核心张量$\{\mathcal{W}^{Re,b}, \mathcal{W}^{Im,b}\}_{b=1}^B$完全固定，所有向量化核心张量均为已知常量
2. **问题转化**：重构损失变为关于潜在函数参数$\theta_x, \theta_y$的非线性优化问题（神经网络为非线性映射），TV正则化项不含$\theta_x, \theta_y$，此步骤仅需优化重构损失
3. **梯度优化**：采用Adam优化器，通过反向传播计算损失关于$\theta_x, \theta_y$的梯度，更新神经网络参数；支持按样本/按频率分batch训练，适配大数据集
4. **输出**：空间潜在函数参数$\theta_x, \theta_y$完成更新

#### 迭代终止条件
重复执行子步骤3.1和3.2，直到满足以下任一条件：
1. 训练损失$\mathcal{L}_{FTM}$连续10轮以上的下降幅度低于预设收敛阈值（如1e-5）
2. 达到预设的最大迭代轮数
3. 潜在函数参数与核心张量的更新量低于预设阈值
#### 实际使用发现，直接联合优化参数，速度更快，效果也很好
### 步骤4：训练收敛后的输出结果
FTM训练完成后，输出两个核心结果，支撑后续频率域生成与重建任务：
1. **固定共享的空间潜在函数**：训练完成的$f_{\theta_x}(x), f_{\theta_y}(y)$，永久固定不再更新，学习到了该类物理场的通用空间坐标表示
2. **训练集样本的核心张量序列**：每个样本对应的实部/虚部核心张量序列$\{\mathcal{W}^{Re,b}, \mathcal{W}^{Im,b}\}_{b=1}^B$，是对应扫频样本的紧凑潜在表示，可直接作为后续生成模型的训练数据

## 三、全场重建解码流程（配套使用）
训练完成后，给定任意频率$\omega_m$的核心张量$\mathcal{W}_{\omega_m}^{Re}, \mathcal{W}_{\omega_m}^{Im}$，可通过FTM解码公式，计算**任意连续空间坐标$(x,y)$**处的复场值，实现从核心张量到全空间二维复场的重建：
$$
\hat{Re}(x,y,\omega_m) = vec(\mathcal{W}_{\omega_m}^{Re})^{\top} \cdot \left( f_{\theta_x}(x) \otimes f_{\theta_y}(y) \right)
$$
$$
\hat{Im}(x,y,\omega_m) = vec(\mathcal{W}_{\omega_m}^{Im})^{\top} \cdot \left( f_{\theta_x}(x) \otimes f_{\theta_y}(y) \right)
$$
最终复场为：$\hat{u}(x,y,\omega_m) = \hat{Re}(x,y,\omega_m) + i \cdot \hat{Im}(x,y,\omega_m)$
